# Hospital Details Dashboard – Power BI

## Overview
This project presents a **Hospital Details Dashboard** built in **Power BI** to analyze and monitor important hospital-related data such as patient details, admission records, doctor information, and overall hospital performance.  
The dashboard helps in understanding key operational insights and supports better decision-making through interactive visualizations.

## Objective
The main objective of this project is to create an interactive dashboard that provides a clear view of hospital data and helps track important metrics related to patients, doctors, admissions, and treatment details.

## Tools Used
- Power BI
- Power Query
- DAX
- Microsoft Excel / CSV Dataset

## Key Features
- Patient details analysis
- Admission and discharge tracking
- Doctor-wise information and performance view
- Department-level insights
- Interactive filters and slicers
- KPI-based reporting for better decision-making

## Dashboard Insights
- Total number of patients
- Admission trends over time
- Department-wise patient distribution
- Doctor-related analysis
- Treatment and hospital activity overview

## Process
1. Collected and imported raw hospital data into Power BI.
2. Cleaned and transformed the data using Power Query.
3. Structured the data model for better reporting.
4. Created DAX measures for KPI calculations.
5. Designed an interactive dashboard using charts, cards, slicers, and tables.

## Outcome
This dashboard helps users quickly understand hospital operations, identify trends, and monitor important healthcare-related metrics in a simple and visual format.
